# statistical-analysis-python

## Description
A Python program that analyzes user-entered numbers by calculating frequency, sum, average, maximum, and minimum values.

## Features
- Accepts integer input from the user.
- Stores numbers in a list.
- Calculates the frequency of each number using a dictionary.
- Computes:
  - Maximum value
  - Minimum value
  - Total sum
  - Average
- Handles invalid input using exception handling.

## Technologies Used
- Python
- Google Colab
